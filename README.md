# NDTV-ui-ux-redesign
A modern UI/UX redesign of the NDTV news website created in Figma for a hackathon. Focused on improving usability, personalization, and the news reading experience.

* Project Overview
Many news websites contain a large amount of information on the homepage, which can make navigation and reading difficult for users. This redesign focuses on simplifying the layout, improving content hierarchy, and enhancing the overall news reading experience.

* Design Tool : Figma

## Key Improvements:
- Cleaner and more organized homepage layout
- Personalized news feed based on user interests
- Improved article reading experience with better typography and spacing
- Simplified navigation for easier access to news categories
- Modern and responsive UI design suitable for desktop and mobile users
